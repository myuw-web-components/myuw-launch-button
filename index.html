Launch Button: specifying only url and aria-label

When accepting the default visual button label, a more specific aria-label will always be needed, so that a screen reader user does not face a page full of links labeled "Launch full app".

Launch Button: also specifying button-text

Some button text may work fine for visual browser users apprehending the launch button in the context of its widget, but still needs further clarified by an aria-label to make the button understandable in isolation.

Launch Button: using button-text as effective ARIA label

Where the button-text sufficiently clarifies what will happen when the user activates the launch button, no explicit aria-label is needed.
